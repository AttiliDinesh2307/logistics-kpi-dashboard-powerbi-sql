# 📦 Logistics Performance Dashboard | Power BI, SQL & DAX

## 📖 Project Overview

The **Logistics Performance Dashboard** is an end-to-end Business Intelligence project developed using **Power BI, SQL (SQLite), DAX, and Excel**. It provides comprehensive insights into sales performance, customer behavior, logistics efficiency, warehouse operations, and financial performance.

The dashboard enables decision-makers to monitor key performance indicators (KPIs), identify operational bottlenecks, and make data-driven business decisions through interactive visualizations.

---

# 🎯 Project Objectives

- Monitor key logistics and operational KPIs
- Analyze sales and customer performance
- Evaluate delivery efficiency and SLA compliance
- Track warehouse utilization and financial performance
- Present actionable business insights using interactive dashboards

---
